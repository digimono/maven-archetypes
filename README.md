## Generate project

```bash
mvn archetype:generate \
    -DarchetypeGroupId=io.github.openfrog \
    -DarchetypeArtifactId=application-archetype \
    -DarchetypeVersion=0.0.1 \
    -DarchetypeCatalog=local \
    -DinteractiveMode=false \    
    -DgroupId=io.github.openfrog \
    -DartifactId=sample
```
